Import photos from a flickr rss feed to a wordpress widget.

== Description ==

The plugin fetch a rss feed of a flickr photostream and show the images in a sidebar widget. You can activate a
lightbox feature if lightbox is installed.

== Installation ==

1. Upload the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the design tab and drag&drop the widget to the right place
4. Enter the title, amount of images and the rss url for working
